# UFG - Disciplinas de Programação

Este repositório reúne os códigos, exercícios e implementações desenvolvidos durante a graduação na **Universidade Federal de Goiás (UFG)**.

## 📂 Estrutura do Repositório

Cada pasta corresponde a uma disciplina. Clique para ver o README detalhado com explicações e instruções de como rodar.

| Pasta | Disciplina | Linguagem | Conteúdo |
|-------|------------|-----------|----------|
| [IP/](IP/) | Introdução à Programação | C | Fundamentos de lógica: entrada/saída, condicionais, laços, funções, vetores e matrizes. |
| [AED1/](AED1/) | Algoritmos e Estruturas de Dados 1 | C | Alocação dinâmica, listas, pilhas, filas, ordenação, busca e recursividade. |
| [AED2/](AED2/) | Algoritmos e Estruturas de Dados 2 | C | Árvores binárias de busca e árvores AVL (projetos em equipe). |
| [POO/](POO/) | Programação Orientada a Objetos | Java | Classes, herança, polimorfismo, interfaces, coleções, arquivos e testes (JUnit). |

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** C e Java
* **Compiladores:** GCC e JDK (javac)
* **Ferramentas:** VS Code

## 🚀 Como Começar

Clone o repositório:

```bash
git clone https://github.com/rafaelaugustodev/UFG.git
cd UFG/Materias
```

As instruções de compilação e execução variam por linguagem. Consulte o README de cada disciplina:

* Disciplinas em **C** (IP, AED1, AED2): exigem o compilador `gcc`.
* Disciplina em **Java** (POO): exige o `javac` e a `java` (JDK).
